Define the Cargo Holds

Set the global download directory and one main library root. Category-specific folders are optional overrides saved to ignored config/categories/<category_id>.yaml files. If a category override is blank, LJS creates and uses <library root>/<category folder>.

How it works: LJS downloads torrent chunks into the Download Directory. Once finished, it verifies the payload and organizes it into the category's effective library folder. Most users only need the global library root; use category overrides only for different disks, shares, or existing Plex/Emby/Jellyfin libraries.

Blank category overrides use this root, for example {{ settings.library_root or './library' }}/Music, {{ settings.library_root or './library' }}/Ebooks, and {{ settings.library_root or './library' }}/Audiobooks.
